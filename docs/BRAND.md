# Spora — Бренд-гайд

Концепция: **тёплый лес после дождя**. Кремовые и землистые тона, приглушённый мох-акцент. Никакого глянцевого медицинского блеска.

## 2. Типография

Все шрифты — бесплатные, с полной поддержкой кириллицы.

| Роль          | Шрифт            | Размер / вес                                                   |
|---------------|------------------|----------------------------------------------------------------|
| Brand / Logo  | **Fraunces**     | 28pt SemiBold — только для названия «Spora» и онбординга       |
| Title 1       | **Onest**        | 22pt Bold                                                      |
| Title 2       | **Onest**        | 18pt Bold                                                      |
| Title 3       | **Onest**        | 15pt SemiBold                                                  |
| Body          | **SF Pro Text**  | 15pt Regular (системный)                                       |
| Caption       | **SF Pro Text**  | 12pt Regular, Text Secondary                                   |

**Источники:**

- Fraunces — Google Fonts, OFL.
- Onest — Google Fonts, OFL.
- SF Pro — системный, не требует подключения.

## 3. Формы и пластика

- **Радиусы скругления:**
  - Карточки: `16pt`
  - Кнопки primary: полная пилюля (`height / 2`)
  - Поля ввода: `12pt`
  - Чипы/теги: полная пилюля
- **Тени:** мягкие, тёплые — `rgba(43,31,23,0.08)` (лёгкий коричневый подтон вместо чёрного).
- **Spacing:** базовая сетка (4 / 6 / 12 / 18 / 24).
- **Кнопки primary:** заливка Primary, мягкая тень, полная пилюля, высота 52pt, текст SF Pro 15pt SemiBold.
- **Анимации:** мягкие, `.spring(response: 0.4, dampingFraction: 0.8)` — без резких переходов.

## 4. Иконография

- Стиль: **outline + filled-active** (контурные иконки в неактивном состоянии, заливка Primary в активном).
- Источник: **SF Symbols** (системные, бесплатные, единый стиль с iOS) — для MVP этого достаточно.
- Иллюстрации БАДов: акварельные / гуашевые рисунки грибов в палитре бренда. На этапе финализации каталога — заказ иллюстратору либо AI-генерация в едином стиле.

## 5. Тон и язык

- Дружелюбный, спокойный, без медицинской сухости и без излишней «бодрости».
- Обращение к пользователю на «вы».

| ❌ Не пишем                    | ✅ Пишем                                  |
|-------------------------------|-------------------------------------------|
| Примите 500 мг чаги!          | Время чаги. С едой.                       |
| Вы пропустили приём!          | Сегодняшний приём остался незамеченным.   |
| Лекарство                     | БАД / приём                               |
